PyRVA: a Python Usergroup

The Python language is a rarity in the software industry: accessible to almost everyone, versatile across application domains and generally a joy to work with. While enjoying Python alone is beneficial, PyRVA believes that Python is best enjoyed in the company of others. We are busy organizing talks, hackathons, workshops and more to help build up the Python community here in Richmond. Ideas are welcome.


Follow us on Twitter, GitHub, and Meetup.

Next meeting: the Depot

Wednesday, November 11, 2015; 6:30–9:30

For our second meetup, we are going to experiment a bit. For the first part of the meetup (~45 mins), we are going to host two workshops: For those of us new to Python, we're running a Python Bootcamp to install Python & Python tools on their machines, including tools like virtualenv and pip. If we have time, we may run through a quick web app demo for fun. For those who have a more extensive Python background, during this time we're going to try "Project Time" that we're borrowing from other Python User Groups like Boston Python and DC Python. Project Time is unstructured time for anyone to work on any projects they'd like, potentially with other PyRVA members.

Following the Python Bootcamp/Project Time, we're going to have a few Lightning Talks. Topics (that we know of!) include Continuous Integration & Python Application Deployment w/ Jenkins; and Metadata Preprocessing in Python and Document Tree Structure. If you have an idea you'd like to present for a 5-15 minute Lightning Talk, you can let Andrew or Chris know, or just show up prepared and we'll schedule you toward the end.
