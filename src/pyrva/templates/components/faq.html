{# Frequency Asked Questions component This uses some more advanced Jinja features to create a reusable FAQ component. This component is used to display a list of questions and answers. Each question is a details element that can be toggled open and closed. The answers are hidden until the question is clicked. You have to "call" the macro with the parameters. By creating a call context, the content in side the call block will be rendered inside the macro. This allows you to pass in the content for each question and answer. https://jinja.palletsprojects.com/en/3.0.x/templates/#call #} {% macro faq(question, id = '') %}
{{ question }}

Frequently Asked Questions

{% call faq('When does PyRVA meet?') %}

PyRVA currently meets on the third tuesday of each month at 5:45 pm. Our meetings may occasionally be rescheduled. Follow us on Meetup to make sure you don't miss any meetings.

{% endcall %} {% call faq("I'd like to present at an upcoming meeting, who should I talk to?", id="present") %}

PyRVA would love to have you present at one of our meetings! Please reach out to us on Discord or contact one of the organizers directly.

{% endcall %} {% call faq('I see you have a YouTube channel, are meetings recorded?') %}

In the past, PyRVA has recorded meetings and posted them to our YouTube channel. However, we do not record our meetings by default as we found it detracted from the in-person experience.

{% endcall %} {% call faq('I am presenting at an upcoming meeting, can my talk be recorded?') %}

While we do not record meetings by default, we are happy to record your talk if you would like. Please reach out to us on Discord. You may also contact one of the organizers directly.

{% endcall %} {% call faq('How do I get involved with PyRVA?') %}

We are always looking for new members to help us grow. You can join our meetings, join our Discord Server, or help us organize events. If you are interested in helping us organize events, please reach out to us on Discord.

{% endcall %} {% call faq('How do I report a Code of Conduct violation?') %}

All members are expected to adhere to the Python Software Foundation's Code of Conduct. If you see a violation, please report it to any organizers. You may also report it to the Python Software Foundation.

{% endcall %} {% call faq('Can I get help with setting up Python on my machine?') %}

Absolutely! Feel free to come early to any of our meetings. Any organizers or senior memeber will be happy to help you. You may also join our Discord Server and ask for help there.

{% endcall %}
